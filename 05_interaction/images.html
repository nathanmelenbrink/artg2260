Rectangles vs. Ellipses

Before loading images, let's use rect() as a placeholder. We know we'll need to write a different bounce() function, since the dimensions on the rectangle are determined by width and height, and not a single diameter. Similarly, we'll need to change the overlap() function. Also, by default rect() draws from the top left corner point of the rectangle, not from the center. We can change that using rectMode(CENTER), but let's not do that for now.

Add Images

This uses what's called string concatenation. In JavaScript, adding 5 + 4 = 9, but adding "five" + "four" = "fivefour". In other words, when you try to add strings, JavaScript merges them into a single string. We can use this feature to create a series like 'assets/0.png', 'assets/1.png', 'assets/2.png', etc.
